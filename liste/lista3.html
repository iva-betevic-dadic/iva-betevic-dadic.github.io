Vjezba 9 - definicijska lista

U ovoj vjezbi koriste se tagovi h1, p, dl, dt i dd. Tag h1 sluzi za glavni naslov stranice. Tag p sluzi za paragraf teksta. Tag dl oznacava definicijsku listu. Tag dt oznacava pojam koji se definira, a tag dd oznacava njegovo objasnjenje. Definicijska lista koristi se kada zelimo prikazati pojam i njegovo znacenje, pa je zato pogodna za rijecnike, pojmovnike i kratka tumacenja izraza.

html
Jezik za izradu strukture web stranice.
head
Dio HTML dokumenta u kojem se nalaze podaci o stranici.
body
Dio HTML dokumenta u kojem se nalazi sadrzaj koji se prikazuje na stranici.
a
Tag koji sluzi za stvaranje poveznica.
